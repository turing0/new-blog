## Design inspired from

- [Lee Robinson](https://leerob.io) - base repository.
- [Pedro Duarte](https://ped.ro) - background grainy/noise effect.
- [Delba](https://delba.dev) - post likes button and table of content.
- [Achintya Jha](https://achintyajha.com/) - mobile menu sidebar.
- [Harsh Singh](https://hxrsh.in)
- and other awesome creators.

## Technologies used

- [Nextjs](https://nextjs.org) - React Framework
- [Typescript](https://www.typescriptlang.org/) - Language that builds on JavaScript.
- [Contentlayer](https://contentlayer.dev) - Mdx Content for posts, snippets and other pages.
- [Tailwindcss](https://tailwindcss.com/) - Styling
- [NextAuth.js](https://next-auth.js.org/) - Authentication
- [Planetscale](https://planetscale.com/) - Database
- [Prisma](https://prisma.io/) - ORM
- [Vercel](https://vercel.com/) - Hosting
- [SVG](https://www.svgrepo.com/) - SVG 
- and many more [packages](https://github.com/heykapil/new-blog/blob/main/package.json).
